Farben

Um das Aussehen verschiedener Gegenstände beschreiben zu können benötigt man meist Farben. Dieser Abschnitt hat verschieden Farben übersetzt um diese auswendig zu lernen. Hier lernt man wie man z.B. hellblau oder dunkelrot auf englisch äussern kann.

- Dieses Thema ist ganz einfach. Wenn man etwas als hell beschreiben möchte sagt man: «This is a very nice light blue.» («Das ist ein sehr schönes hellblau.»)

- Es kann auch bright als hell benutzt werden. Hier ist ein Beispiel: « These are pretty bright green shoes.» («Das sind schöne hellgrüne Schuhe.»)

- Möchte man sagen, dass eine Farbe sehr dunkel ist sagt man folgendes: « This dark blue dress suits you well.» («Das dunkelblaue Kleid steht dir sehr gut.»)

- Man kann matt so benutzen auf englisch: « This is a dullgreen jacket.» («Das ist eine mattgrüne Jacke.»)
